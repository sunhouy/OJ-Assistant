Educoder助手用户手册

欢迎使用Educoder助手，本手册将指导您完成软件安装、注册登录及功能使用等操作

登录与注册

登录：打开软件后，您将看到登录界面。请输入您的用户名和密码进行登录。

登录界面示例图

注册：新用户需要注册账号。注册时，需要输入邀请码。成功注册后，邀请人和被邀请人均免费获得一天会员。

注册界面示例图

重要规则：

  • 同一设备上，注册的第一个账号输入邀请码可获得会员，第二个及之后的账号即使输入邀请码也不再赠送会员。
  • 用户名与设备ID唯一绑定，一个账号只能在注册时的设备上使用。
  • 此机制防止同一个人开通会员后将账号分享给多人使用。

安装浏览器扩展

初次注册后，软件会提示您安装浏览器扩展。扩展会自动安装到Microsoft Edge浏览器。

扩展安装提示

安装步骤会显示在界面左上角，您可以查看安装进程。

安装步骤示意图

注意：扩展安装后，当您直接启动Edge时，扩展可能无法正确显示。建议每次通过软件主界面的"启动浏览器"按钮来启动浏览器，以确保扩展正常工作。

手动安装：如果您需要永久、稳定地安装扩展，或需要在Chrome浏览器上安装，请点击主界面的"启动浏览器"按钮，然后点击"手动安装"按钮查看详细教程。

启动与连接

登录成功后，您需要先开通会员才能启动服务器。启动服务器的步骤如下：

  1. 1. 确保您已开通会员
  2. 2. 在客户端启动服务器
  3. 3. 浏览器插件即可使用（插件默认隐藏）

浏览器插件使用方式：

  • 按下 F4 显示浏览器插件的主界面
  • 按下 F1 开始自动答题
  • 按下 F2 智能纠错
  • 按下 F3 尝试连接服务器
浏览器插件主界面
连接服务器界面

您也可以通过刷新网页自动连接服务器。

主界面功能

软件主界面提供丰富的功能，帮助您更好地使用Educoder助手。

软件主界面

免费输入测试工具

输入测试工具界面

这个工具可以在所有阻止复制粘贴的浏览器网页或桌面客户端上进行模拟键盘输入，确保输入的准确性。

模型选择

您可以选择合适的AI模型，也可以自己添加模型。对于自己添加的模型，无需开通会员即可启动服务器。

提示：您可以在一些AI模型的开放平台注册开发者账号以获得API Key，然后将模型添加到软件中使用。

启用复制粘贴模式

此功能仅在编辑器允许复制粘贴时可用。它可以一次性输入完整代码，大大提高输入速度。

说明：虽然我们的模拟键盘输入已经非常快速和准确，但我们仍然保留了复制粘贴模式，以满足不同场景的需求。

编程语言选择

您可以在选择编程语言区域指定需要在编辑器中输入的编程语言。软件提供的编程语言都经过充分测试，能够保证准确性。

暂不支持添加自定义编程语言，以防止未知问题的发生。

其他设置与功能

显示日志

此功能仅在遇到问题需要反馈时使用。您可以将日志提供给开发者，帮助诊断和解决问题。

开机自启

开启此功能后，软件会在每次开机时自动启动。请注意，部分杀毒软件可能会阻止此功能。

关闭时最小化到托盘

如果勾选此选项，当您关闭应用时，它不会完全退出，而是最小化到桌面右下角的系统托盘。您可以在托盘图标上点击右键，选择恢复主界面。

启动浏览器按钮

建议始终通过主界面的"启动浏览器"按钮来启动浏览器，以确保浏览器扩展正常工作。
